### unsafeForceValue

▸ **unsafeForceValue**(`value`): `void`

THIS IS NOT RECOMMENDED - forcing the wheel value can cause issues,
namely the shuffle queue will not know of this. Make sure you know
what you're doing.

**`example`**
```js
wheel.unsafeForceValue("something dumb");
```
