# Class: Spin<T\>

An ongoing spin of a set of simulated wheels. Responsible
for handling the variance in physics across the wheels, if any.

### constructor

• **new Spin**<`T`\>(`parameters`)

The basic constructor.

**`example`**
```js
const spin = new Spin({
  id: "myID",
  wheels: new Map([
    ["emotion", ["sad", "happy"]],
    ["color", ["red", "blue"]],
    ["length", ["long", "short"]]
  ]),
  physics: {
```
